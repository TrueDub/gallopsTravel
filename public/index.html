Real-time Travel Information for The Gallops

This site gives real-time travel information on Luas trams and Dublin buses for all stops along Murphystown Way and Ballogan Road, Dublin 18.

All data is sourced from the Luas Real-Time API and the Dublin Bus RTPI API.

This site just reports the data we're given, if it's wrong it's not our fault!

Should you have any suggestions on improvements to this site, please email us on gallopstravel@gmail.com.
